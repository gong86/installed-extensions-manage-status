# Installed Extensions Manage Status

Browse installed VS Code extensions in a custom sidebar with:
- extension-pack grouping
- installed vs built-in counts
- enabled vs disabled badges
- quick Manage handoff to the native Extensions view

## Features

- Activity Bar view container
- Webview card layout
- Grouping by installed extension packs
- Summary cards for total, installed, and built-in counts
- Manage, Copy ID, Copy Install Command, and Marketplace actions
